# Hero Electricals Landing Page - PRD

## Problem Statement
Build a modern, interactive landing page for Hero Electricals - a retail and wholesale shop for home electrical products (wires, cables, switches, LEDs, switchboards, sockets, connectors). Clean & minimal aesthetic with interactive electrical-themed UI elements.

## Architecture
- **Frontend**: React + Tailwind CSS + Framer Motion + react-fast-marquee
- **Backend**: Not used (landing page only)
- **Database**: Not used

## User Personas
- **Retail Customers**: Homeowners looking for quality electrical products
- **Wholesale Buyers**: Contractors, builders, electricians needing bulk orders

## Core Requirements
- Interactive switchboard toggle (retail/wholesale)
- LED tubelight scroll animation
- Socket-themed contact reveal
- Bento grid product categories
- Brand marquee
- Responsive design
